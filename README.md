## I18n support

add `i18n` field in your resume.

e.g. if I want some label show in zh-CN.

```json
{
    "i18n": {
		"Contact": "联系方式",
		"About": "自我介绍",
		"Profiles": "社交账号",
		"Work": "从业经历",
		"Education": "教育",
		"Skills": "技能",
		"Volunteer": "志愿活动",
		"Publications": "发表作品",
		"Awards": "获奖",
		"Languages": "语言",
		"References": "证明人",
		"Interests": "兴趣"
	},
	"basics": {
		"name": "hxsf",
		"label": "Programmer",
```
